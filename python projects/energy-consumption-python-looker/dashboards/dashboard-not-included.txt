DASHBOARD FILES NOT INCLUDED IN THIS REPOSITORY
================================================

The Google Looker Studio dashboard is a cloud-hosted report and is not
stored as a local file in this repository.

The dashboard is built on top of the processed Excel output files
generated by the Jupyter Notebooks in the notebooks/ folder.

To view the dashboard:
- Please contact the author for the Looker Studio report link.
- Refer to the screenshots/ folder for visual previews of each dashboard page.

Dashboard Pages
---------------
1. ESB Energy Insights — KPIs, consumption time-series, day/night tariff split,
   cost breakdown, export revenue, and monthly trend comparison.

2. Solar PV Analysis — Daily generation, grid export, battery flows, full load
   hours, weather correlation (temperature, sunshine, precipitation).
