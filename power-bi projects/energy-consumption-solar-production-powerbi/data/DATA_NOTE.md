# Data

This folder is reserved for source data files used in the project.

## Data Sources

| Source | Description |
|--------|-------------|
| **ESB (Electricity Supply Board)** | Grid electricity import/export data (kWh)
| **SolisCloud** | Solar generation, battery charge/discharge, load consumption, grid import and export |
| **Meteostat API** | mean temp, tmin, tmax, tsun, prcp |

## Privacy Notice

Due to confidentiality, the original datasets are **not included** in this repository. The data contains personally identifiable energy usage and financial information.

For questions about the data structure or methodology, please contact the author.
